# Casos de uso de personalização{#section-overview}

Esta seção fornece casos de uso passo a passo para criar mensagens personalizadas no Adobe Journey Optimizer. Cada exemplo é baseado em um cenário do mundo real e mostra como combinar atributos de perfil, dados contextuais, funções auxiliares e decisões de oferta para fornecer conteúdo direcionado e relevante ao seu público-alvo.

## Casos de uso de personalização

:::: landing-cards-container
:::
![icon](https://cdn.experienceleague.adobe.com/icons/circle-play.svg?lang=pt-BR)

Notificação do status do pedido

Analise um cenário completo que combina atributos de perfil, dados contextuais em tempo real e o Offer Decisioning para enviar aos clientes uma notificação por push personalizada confirmando ou atualizando o status do pedido.

[Leia o tutorial](../using/personalization/personalization-use-case.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/envelope.svg?lang=pt-BR)

Email de abandono do carrinho

Saiba como usar funções auxiliares para preencher dinamicamente um email de abandono de carrinho com detalhes específicos do cliente e informações do produto, ajudando você a recuperar vendas perdidas com mensagens relevantes e oportunas.

[Leia o guia](../using/personalization/personalization-use-case-helper-functions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/list-check.svg?lang=pt-BR)

Alerta de prescrição de plano de saúde

Crie um único email personalizado que lista todas as receitas de um assinante que estão atualmente com o status &quot;pronto para retirada&quot; ou &quot;retomado&quot;, usando lógica condicional e expressões de modelo.

[Leia o exemplo](../using/personalization/perso-uc-plan-prescriptions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/code-branch.svg?lang=pt-BR)

Receitas do Personalization

Navegue em uma biblioteca de padrões de personalização prontos para uso que abrangem datas, atributos de perfil, lógica condicional e muito mais. Cada fórmula inclui um trecho de sintaxe de trabalho que você pode copiar diretamente para seu conteúdo de email, SMS ou push.

[Procurar as receitas](../using/personalization/personalization-recipes.md)
:::

## Recursos adicionais

- **[Visão geral da função indexOf](../using/building-journeys/functions/string-functions.md#indexOf)**: localize a posição de uma substring em uma string.
- **[Referência da função isEmpty](../using/building-journeys/functions/string-functions.md#isEmpty)**: determine se uma string está vazia usando a função “isEmpty”.
- **[Referência da função isNotEmpty](../using/building-journeys/functions/string-functions.md#isNotEmpty)**: verifique se uma string não está vazia usando a função “isNotEmpty”.
- **[Função lastIndexOf](../using/building-journeys/functions/string-functions.md#lastIndexOf)**: localize a posição da última ocorrência de uma substring.
- **[Visão geral da função Length](../using/building-journeys/functions/string-functions.md#length)**: calcule o número de caracteres em uma string.
- **[Uso da função “lower”](../using/building-journeys/functions/string-functions.md#lower)**: converta strings em minúsculas usando a função “lower”.
- **[Referência da função matchRegExp](../using/building-journeys/functions/string-functions.md#matchRegExp)**: verifique se uma string corresponde a uma expressão regular.
- **[Função NotEqualIgnoreCase](../using/building-journeys/functions/string-functions.md#notEqualIgnoreCase)**: determine se as strings são diferentes, ignorando a diferenciação entre maiúsculas e minúsculas.
- **[Visão geral da função replace](../using/building-journeys/functions/string-functions.md#replace)**: substitua a primeira ocorrência de uma string ou padrão por outra string.
- **[Função replaceAll](../using/building-journeys/functions/string-functions.md#replaceAll)**: substitua todas as ocorrências de uma string ou padrão por outra string.
- **[Uso da função split](../using/building-journeys/functions/string-functions.md#split)**: divida uma string em tokens usando a função “split”.
- **[Referência da função startWith](../using/building-journeys/functions/string-functions.md#startWith)**: verifique se uma string começa com um prefixo especificado.
- **[Função startWithIgnoreCase](../using/building-journeys/functions/string-functions.md#startWithIgnoreCase)**: avalie se uma string começa com um prefixo, sem distinção entre maiúsculas e minúsculas.
- **[Visão geral da função substr](../using/building-journeys/functions/string-functions.md#substr)**: extraia substrings usando índices de início e término.
- **[Referência da função trim](../using/building-journeys/functions/string-functions.md#trim)**: remova espaços à esquerda e à direita de strings.
- **[Visão geral da função upper](../using/building-journeys/functions/string-functions.md#upper)**: transforme as strings em maiúsculas usando a função “upper”.
- **[Referência da função uuid](../using/building-journeys/functions/string-functions.md#uuid)**: gere Identificadores Únicos Universais (UUIDs) aleatórios.

# Perfis de produto incorporados {#ootb-product-profiles}

## Administrador do Jornada {#journey-administrator}

O perfil de produto **[!UICONTROL Journey Administrator]** permite acesso aos menus de administração com a possibilidade de gerenciar e publicar Jornadas, mensagens e gerenciamento de decisões.

Este perfil de produto conta com as seguintes permissões:

| Capacidade | Permissões|
|-|-|
|Jornadas| <ul><li> **Gerenciar jornadas**: acesso para ler, criar, editar e excluir jornadas.</li><li>**Publicar jornadas**: acesso para publicar jornadas.</li><li>**Gerenciar eventos do jornada, fontes de dados e ações**: acesso para ler, criar, editar e excluir eventos e fontes e ações.</li><li>**Exibir relatório** de jornadas: acesso para ler, editar, relatório do jornada.</li></ul>|
|Mensagens|<ul><li> **Gerenciar mensagens**: acesso para ler, criar e editar a pré-visualização da mensagem e enviar teste/prova.</li><li>**Gerenciar pré-visualização e teste** de mensagens: acesso para publicar mensagens.</li><li>**Publicar mensagens**: acesso para ler, criar e editar a pré-visualização da mensagem e enviar teste/prova.</li><li>**Exibir relatório** de mensagens: acesso para ler, editar, relatório de mensagens.</li></ul>|
|Administração|<ul><li>**Gerenciar delegação** de subdomínios: acesso para ler, criar, editar e excluir delegação de subdomínio.</li><li>**Gerenciar pools** de IP: acesso para ler, criar, editar e excluir o pool ip.</li><li>**Exibir registros** PTR: acesso somente leitura a registros PTR.</li><li> **Gerenciar configurações** gerais de mensagens: acesso para ler, criar, editar e excluir configurações gerais de mensagens.</li><li>**Gerenciar predefinições** de mensagens: acesso para ler, criar, editar e excluir marcas de conteúdo.</li><li>**Gerenciar regras** de supressão: acesse ler, criar, editar e excluir regras de supressão.</li><li>**Exibir lista** de supressão: acesso para ler e exportar a lista de supressão local.</li><li>**Gerenciar alertas**: acesso para ativar/desativar alertas para jornadas, mensagens e direitos.</li></ul>|
|Gestão das decisões|<ul><li>**Gerenciar decisões**: acesso para ler, criar, editar e excluir decisões.</li><li>**Gerenciar estratégias** de classificação: acesso para ler, criar, editar e excluir o relatório de mensagens personalizadas e usar recursos de ação.</li></ul>|
|Adobe Experience Platform|<ul><li>**Sandbox**: acesso para fornecer acesso a sandboxes.</li><li>**Gerenciar segmentos**: acesso para ler, criar, editar e excluir segmentos.</li><li>**Gerenciar perfis**: acesso para ler, criar, editar e excluir perfis.</li><li>**Conjuntos** de dados de leitura: acesso somente leitura a conjuntos de dados.</li><li>**Ler esquemas**: acesso somente leitura a schemas.</li><li>**Ler namespace** de identidade: acesso somente leitura ao namespace de identidade.</li></ul>|

## Aprovador de jornada {#journey-approver}

O perfil de produto **[!UICONTROL Journey Approver]** permite que os usuários aprovem os deliveries e os publiquem. Posteriormente, eles podem verificar o sucesso de seus deliveries com os relatórios **[!UICONTROL Message]** e **[!UICONTROL Journey]**.

Este perfil de produto conta com as seguintes permissões:

| Capacidade | Permissões|
|-|-|
|Jornadas| <ul><li>**Gerenciar jornadas**: acesso para ler, criar, editar e excluir jornadas.</li><li>**Publicar jornada**: acesso para publicar jornadas.</li><li>**Exibir eventos do jornada, fontes de dados e ações**: acesso somente leitura a eventos do jornada, ações personalizadas do jornada e fontes de dados do jornada.</li><li>**Exibir relatório** de jornadas: acesso para ler, editar o relatório de jornada e usar recursos de ação como compartilhamento de relatório.</li></ul>|
|Mensagens| <ul><li>**Gerenciar mensagens**: acesso para ler, criar, editar e excluir mensagens.</li><li>**Publique** mensagens e acesse para publicar mensagens.</li><li>**Gerenciar pré-visualização e teste** de mensagens: acesso para ler, criar e editar a pré-visualização da mensagem e enviar teste/prova.</li><li>**Exibir relatório** de mensagens: acesso para ler, criar, editar e excluir o relatório de mensagens.</li></ul>|
|Gestão das decisões| <ul><li>**Gerenciar decisões**: acesso para ler, criar, editar e excluir entidades de decisão.</li><li>**Gerenciar estratégias** de classificação: acesso para ler, criar, editar e excluir o relatório de mensagens personalizadas e usar recursos de ação.</li></ul>|
|Adobe Experience Platform| <ul><li>**Gerenciar segmentos**: acesso para ler, criar, editar e excluir segmentos.</li><li>**Gerenciar perfis**: acesso para ler, criar, editar e excluir perfis.</li><li>**Conjuntos** de dados de leitura: acesso somente leitura a conjuntos de dados.</li><li>**Ler esquemas**: acesso somente leitura a schemas.</li></ul>|
|Administração| <ul><li>**Exibir predefinições** de mensagens: acesso somente leitura às predefinições de mensagens.</li></ul>|

|Gestão das decisões| <ul><li>**Gerenciar decisões**: acesso para ler, criar, editar e excluir entidades de decisão.</li><li>**Gerenciar estratégias** de classificação: acesso para ler, criar, editar e excluir o relatório de mensagens personalizadas e usar recursos de ação.</li></ul>|
|Adobe Experience Platform| <ul><li> **Gerenciar segmentos**: acesso para ler, criar, editar e excluir segmentos.</li><li>**Gerenciar perfis**: acesso para ler, criar, editar e excluir perfis.</li><li>**Conjuntos** de dados de leitura: acesso somente leitura a conjuntos de dados.</li><li>**Ler esquemas**: acesso somente leitura a schemas.</li></ul>|
|Administração| <ul><li>**Exibir predefinições** de mensagens: acesso somente leitura às predefinições de mensagens.</li></ul>|

## Visualizador de jornada {#journey-viewer}

O perfil de produto **[!UICONTROL Journey viewer]** permite acesso somente leitura aos recursos **[!UICONTROL Journeys]**, **[!UICONTROL Goals]**, **[!UICONTROL Messages]** e **[!UICONTROL Decision management]**.

Os usuários atribuídos a este perfil de produto não poderão editar ou publicar.

Este perfil de produto conta com as seguintes permissões:

| Capacidade | Permissões|
|-|-|
|Jornadas| <ul><li>**Jornadas** de exibição: acesso somente leitura ao jornada.</li><li>**Exibir evento de jornadas, fontes de dados, ações**: acesso somente leitura a eventos do jornada e fontes de dados.</li><li>**Exibir relatório** de jornadas: acesso somente leitura ao relatório jornada.</li></ul>|
|Mensagens| <ul><li>**Exibir mensagens**: acesso somente leitura à mensagem.</li><li>**Exibir relatório** de mensagens: acesso somente leitura ao relatório de mensagem.</li></ul>|
|Gestão das decisões| <ul><li>**Exibir decisões**: acesso somente leitura às entidades de decisão.</li></ul>|

## Gerenciador de mensagens {#message-manager}

O perfil de produto **[!UICONTROL Message Manager]** permite que os usuários criem e editem **[!UICONTROL Messages]** e **[!UICONTROL Decision management]**, mas não poderá publicá-los.

Este perfil de produto conta com as seguintes permissões:

| Capacidade | Permissões|
|-|-|
|Jornadas| <ul><li>**Jornadas** de exibição: acesso somente leitura ao jornada.</li></ul>|
|Mensagens| <ul><li>**Gerenciar mensagens**: acesso para ler, criar, editar e excluir mensagens.</li><li>**Gerenciar pré-visualização e teste** de mensagens: acesso para ler, criar e editar a pré-visualização da mensagem e enviar teste/prova.</li><li> **Exibir relatório** de mensagens: acesso para ler, criar, editar e excluir o relatório de mensagens.</li></ul>|
|Gestão das decisões| <ul><li>**Gerenciar decisões**: acesso para ler, criar, editar e excluir entidades de decisão.</li></ul>|
|Adobe Experience Platform| <ul><li>**Ler perfis**: acesso somente leitura ao perfil para visualização e teste.</li><li>**Conjuntos** de dados de leitura: acesso somente leitura a conjuntos de dados.</li><li>**Ler esquemas**: acesso somente leitura a schemas.</li></ul>|
|Administração| <ul><li>**Exibir predefinições** de mensagens: acesso somente leitura às predefinições de mensagens de leitura.</li></ul>|

## Gerente de decisão {#decisioning-manager}

O perfil de produto **[!UICONTROL Decisioning manager]** permite acesso somente ao menu **[!UICONTROL Decision management]** . Os usuários atribuídos a este perfil de produto só poderão gerenciar, visualizar e publicar decisões.

Este perfil de produto conta com as seguintes permissões:

| Capacidade | Permissões|
|-|-|
|Gestão das decisões| <ul><li>**Gerenciar decisões**: acesso para ler, criar, editar e excluir entidades de decisão.</li><li>**Exibir decisões**: acesso somente leitura às entidades de decisão.</li><li>**Gerenciar estratégias** de classificação: acesso para ler, criar, editar e excluir o relatório de mensagens personalizadas e usar recursos de ação.</li><li>**Publicar decisões**: acesso às atividades de decisão Aprovar/Cancelar aprovação .</li></ul>|

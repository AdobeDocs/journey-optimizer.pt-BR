# round {#round}

Retorna o valor inteiro mais próximo do argumento com vínculos arredondados para o infinito positivo.

## Categoria

Matemática

## Sintaxe da função

`round(<parameters>)`

## Parâmetros

* decimal
* inteiro

## Assinaturas e tipo retornado

`round(<decimal>)`

`round(<integer>)`

Retorna um número inteiro.

## Exemplo

`round(3.14)`

Retorna 3.

`round(3.54)`

Retorna 4.

`round(-3.14)`

Retorna -3.

`round(3)`

Retorna 3.

# trim {#trim}

Remove espaços de início e fim.

## Categoria

String

## Sintaxe da função

`trim(<parameters>)`

## Parâmetro

| Parâmetro | Tipo |
|-----------|------------------|
| string | string |

## Assinatura e tipo retornado

`trim(<string>)`

Retorne uma string.

## Exemplo

`trim(" Hello ")`

Retorna &quot;Hello&quot;.

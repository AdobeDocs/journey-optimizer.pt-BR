# comprimento {#length}

Retorna o número de caracteres da expressão de cadeia de caracteres no parâmetro.

## Categoria

String

## Sintaxe da função

`length(<parameters>)`

## Parâmetro

* sequência de caracteres

## Assinatura e tipo retornado

`length(<string>)`

Retorna um inteiro.

## Exemplo

`length("Hello World")`

Retorna 11.

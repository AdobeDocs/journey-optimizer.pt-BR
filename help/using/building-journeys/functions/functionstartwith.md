# startWith {#startWith}

Retorna true se o segundo parâmetro for um prefixo do primeiro.

## Categoria

String

## Sintaxe da função

`startWith(<parameters>)`

## Parâmetros

| Parâmetro | Tipo |
|-------------|--------|
| string | string |
| prefixo | string |

## Assinatura e tipo retornado

`startWith(<string>,<string>)`

Retorne um booleano.

## Exemplo

`startWith("Hello World", "Hello")`

Retorna true.

`startWith("Hello World", "World")`

Retorna false.

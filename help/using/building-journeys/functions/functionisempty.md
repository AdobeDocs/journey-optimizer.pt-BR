# isEmpty {#isEmpty}

Retornará true se a cadeia de caracteres no parâmetro não tiver caractere.

## Categoria

String

## Sintaxe da função

`isEmpty(<parameters>)`

## Parâmetros

* sequência de caracteres

## Assinatura e tipo retornado

`isEmpty(<string>)`

Retorna um valor booleano.

## Exemplo

`isEmpty("")`

Retorna verdadeiro.

`isEmpty("Hello World")`

Retorna falso.

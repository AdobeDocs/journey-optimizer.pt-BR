# upper {#upper}

Retorna uma versão em maiúsculas do parâmetro .

## Categoria

String

## Sintaxe da função

`upper(<parameters>)`

## Assinatura e tipo retornado

`upper(<string>)`

Retorne uma string.

## Exemplo

`upper("b")`

Retorna &quot;B&quot;.

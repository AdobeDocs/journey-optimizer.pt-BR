>[!BEGINSHADEBOX]

**Nesta página:** saiba como usar a atividade End para marcar o final de cada caminho de jornada e adicionar rótulos que facilitem a leitura dos relatórios de jornada.

A atividade **[!UICONTROL End]** permite marcar o final de cada caminho da jornada. Não é obrigatório, mas recomendado para clareza visual. Na verdade, se a jornada tiver várias atividades finais, recomendamos adicionar um rótulo a cada “Fim” para facilitar a leitura dos relatórios. Consulte [esta página](../reports/live-report.md).

* **TL;DR:** esta página explica a atividade End no Journey Optimizer, um marcador visual colocado no final de cada caminho de jornada que é opcional, mas recomendado para facilitar a leitura do relatório.

**Intenções:**

* Adicione uma atividade End para marcar a conclusão de um caminho de jornada
* Atividades de Fim de rótulo para melhorar a legibilidade do relatório do jornada
* Distinguir visualmente vários caminhos de jornada usando nós Finais rotulados

**Glossário:**

* **Encerrar atividade**: um nó de jornada que marca visualmente o término de um caminho de jornada; não é obrigatório, mas auxilia na clareza *(específico do produto)*

**Medidas de Proteção:**

* A atividade End não é obrigatória; as jornadas funcionam sem ela.
* É recomendável adicionar um rótulo a cada atividade End quando uma jornada tem vários caminhos, para facilitar a interpretação dos relatórios em tempo real.

**Terminologia:**

* Nome canônico: Finalizar atividade — Acrônimo: n/a — variantes: Finalizar nó, Finalizar tag
* Não confunda: &quot;End activity&quot; ≠ &quot;End tag&quot; — A tag End é um nó gerado automaticamente visível durante a criação do jornada, enquanto a atividade End é um componente posicionável

**Perguntas frequentes:**

* **P: A atividade Fim é necessária para publicar uma jornada?** — Não, não é obrigatório, mas é recomendado para maior clareza visual.
* **P: Por que devo rotular atividades End?** — Quando uma jornada tem vários caminhos, os rótulos ajudam a diferenciá-los nos relatórios de jornada.
* **P: Onde posso ver o impacto dos rótulos de atividade de Fim?** — Os rótulos são exibidos no relatório de jornada em tempo real, facilitando o rastreamento do desempenho em nível de caminho.

# Introdução a consultas {#queries-gs}

O Editor de consultas é uma ferramenta interativa fornecida pelo Serviço de consultas da Adobe Experience Platform que permite gravar, validar e executar consultas para dados de experiência do cliente na interface [!DNL Adobe Journey Optimizer].

O Editor de consultas oferece suporte ao desenvolvimento de consultas para análise e exploração de dados e permite que você execute consultas interativas para fins de desenvolvimento, bem como consultas não interativas para preencher [conjuntos de dados](get-started-datasets.md).

![](assets/queries-home.png)

Saiba como usar o Editor de consultas [nesta documentação](https://experienceleague.adobe.com/docs/experience-platform/query/ui/user-guide.html?lang=pt-BR){target="_blank"}.

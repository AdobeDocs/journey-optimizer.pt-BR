# Adicionar atributos aos favoritos {#fav}

>[!BEGINSHADEBOX]

**Nesta página:** saiba como adicionar, acessar e remover atributos no menu de favoritos para reutilização rápida ao criar expressões de personalização no Adobe Journey Optimizer.

>[!ENDSHADEBOX]

Adicionar atributos diferentes ao menu de favoritos fornece acesso rápido aos itens usados com mais frequência. Para adicionar um atributo aos favoritos, clique no menu de reticências e escolha **[!UICONTROL Adicionar aos favoritos]**.

<!--
![](assets/favorite-option.png)
-->

Para acessar os itens que você marcou como favoritos, use o menu **[!UICONTROL Favoritos]** no painel esquerdo.

Nessa lista, é possível adicionar rapidamente o objeto de personalização à expressão atual.

<!--
![](assets/favorite-list.png)
-->

Se não quiser mais ver um item na lista de favoritos, você pode removê-lo dos favoritos.

<!--
![](assets/favorite-remove.png)
-->

**TL;DR**

Esta página explica como adicionar atributos usados com frequência ao menu Favoritos no editor de personalização, acessá-los para reutilização rápida e removê-los quando não forem mais necessários.

**Intenções**

* Adicionar um atributo de personalização ao menu Favoritos para acesso rápido
* Acesse os atributos favoritos no menu Favoritos no painel esquerdo do editor
* Remover um atributo da lista Favoritos

>[!TAB Glossário]

* **Menu de favoritos**: uma seção dedicada no painel de navegação esquerdo do editor de personalização que fornece acesso rápido aos atributos que o usuário marcou como favoritos, permitindo uma criação de expressão mais rápida. *(específico do produto)*

>[!TAB Terminologia]

* **Nome canônico:** Favoritos — variantes: menu favoritos, atributo favoritos, atributos favoritos
* **Nome canônico:** Favoritos — variantes: menu favoritos, atributos favoritos

>[!TAB Perguntas frequentes]

**P: Como adicionar um atributo aos favoritos?**

Clique no menu de reticências ao lado do atributo no painel de navegação e escolha **Adicionar aos favoritos**.

**P: Onde encontro meus atributos favoritos?**

No menu **Favoritos**, no painel esquerdo do editor de personalização.

**P: Como remover um atributo dos favoritos?**

No menu **Favoritos**, no painel esquerdo, remova o item dos favoritos.
